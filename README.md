# EFR32_Trust-M
Project involving the creation of drivers between the Silicon Labs' Microcontroller EFR32 and Infineon's [OPTIGA&trade; Trust M](https://www.infineon.com/cms/en/product/security-smart-card-solutions/optiga-embedded-security-solutions/optiga-trust/optiga-trust-m-sls32aia/) using I/O Stream over USART.
